REACT DASHBOARD
A web application written in React + Flux + ExpressJS

1) - git clone https://github.com/Giorgio-IMAGINARIA/react-dashboard.git
2) - type "npm install" in the root directory
4) - type "npm run start:prod" in the root directory
5) - navigate to "http://localhost:3001/" in your browser
6) - enjoy :)

Also,

8) - check the __test__ directories at /source/library/ and /components/ (the latter must be improved)
7) - run tests with Jest (npm test)
